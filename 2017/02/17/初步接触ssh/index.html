Posted on 2017-02-17 | In Java进阶 , 工作笔记

2017.2.6
刚到公司，装了环境，了解了项目团队，接触了项目部分代码，学习了新知识。

任务是：将数据库中指定数据显示到页面中。

最初十分迷惑，由于不懂ssh框架，不懂action、service、dao层的关系和作用，不懂

最大的收获是，硬着头皮把压根不懂得ssh框架看的有了点思路。

【 Java Web中的Action、Dao、Service、Model学习笔记—–阿冬专栏

http://blog.csdn.net/zhangdong305/article/details/48048003 】

数据传输过程：Jsp→Action→Service→Dao→Database

配置文件：

web.xml：配置引用的applicationContext文档

struts.xml：网页中可通过name调用相应的action

applicationContext.xml：配置数据库，以及sessionFactory

applicationContext-struts2.xml：配置各个bean对应的类地址，指定调用的service

applicationContext-service.xml：配置各个service对应的类地址，指定调用的dao

applicationContext-hibernate.xml：配置各个dao对应的类地址，指定调用的sessionFacory
