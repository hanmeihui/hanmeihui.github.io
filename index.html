一.FusionCharts的分类

1.FusionCharts。跨平台、跨浏览器的JavaScript/HTML图表组件

FusionCharts XT作为FusionCharts图表套包的主打产品，它提供了所有通用的图表类型，同时它还支持 ASP、ASP.NET、 PHP、 JSP、 ColdFusion、 Ruby on Rails、JavaScript、甚至简单的HTML页面。Fusioncharts来制作专业的JavaScript图表。

2.FusionMaps。

FusionMaps是一款JavaScript (HTML5) &Flash地图工具。可以根据内容、地区来展现地理数据或者其他实体。你可以使用FusionMaps有效地描绘业务数据，例如地区利润、地区人口分布情况、竞选结果、航班路线、办公地点、调查结果等。

3.FusionWidgets。

FusionWidget是所有图表完美的附加组件，即时股票监视器，对KPI和其他重要数据进行实时监视，处理监控应用程序和财务应用程序。通过使用各种易于操作的组件，它将带给你一个程序员和信息可视化相结合的体验。借助JavaScript (HTML5)和Flash, FusionWidgetsXT可在PCs、Macs、iPads, iPhones以及其他移动设备上无缝操作。

4.PowerCharts。适用于专业领域的高级图表控件。

PowerCharts XT是一套用在特定领域如：网络图、盈亏分析、财务规划、股价图以及层次结构图的高级图表库。它还提供了视觉上可编辑的图表来模拟假设情景和雷达图表上的多个实体的多个参数间的比较。

FusionCharts的FusionCharts分类有很多种。

sql server知识点

Posted on 2017-03-04 | In 工作笔记
1、【Order By不能用在派生表中】

表子查询(也叫派生表)中代表的是一个关系表，而关系表中的行是没有固定顺序的。

因此，ANSI SQL不允许在表子查询中包含order by子句。

SQL Server大致遵守了这一规定，但有两个例外：

当表子查询中有top或for xml子句时，可以包含order by子句。此时order by子句仅仅是为了筛选数据行，并不用于展示的目的。因此，子查询的结果不能保证是有序的。

2、【查询指定日期指定关键词的数据以及之前的5条数据，最终结果按日期升序排列】
